Kebijakan Privasi untuk Game "Game Console City"

Diperbarui pada: 16 Oktober 2024

Kami di Rajasa Digital menghargai privasi Anda dan berkomitmen untuk melindungi data pribadi pengguna kami. Kebijakan Privasi ini menjelaskan bagaimana kami mengelola informasi yang terkait dengan penggunaan game "Game Console City". Kami ingin menegaskan bahwa game ini tidak mengumpulkan, menyimpan, atau memproses data pribadi dari pengguna.

1. Pengumpulan Data Pengguna

Game "Game Console City" tidak memerlukan informasi pribadi apa pun dari pengguna untuk dimainkan. Kami tidak meminta pengguna untuk mendaftar, login, atau memberikan data identitas, seperti nama, alamat email, atau informasi lainnya.

2. Data yang Tidak Kami Kumpulkan

  • Data Pribadi: Kami tidak mengumpulkan informasi pribadi pengguna, seperti nama, alamat, nomor telepon, atau email.
  • Data Lokasi: Kami tidak mengumpulkan informasi terkait lokasi geografis pengguna.
  • Data Perangkat: Kami tidak mengumpulkan informasi teknis terkait perangkat pengguna, seperti model perangkat, sistem operasi, atau pengenal perangkat.

3. Iklan dan Pembelian Dalam Aplikasi

Game "Game Console City" tidak menampilkan iklan dan tidak memiliki pembelian dalam aplikasi (In-App Purchases). Pengalaman bermain Anda bebas dari gangguan iklan atau permintaan pembelian tambahan.

4. Data Penggunaan dan Statistik

Kami tidak mengumpulkan data penggunaan atau statistik permainan. Semua aktivitas dalam game berjalan secara lokal di perangkat Anda dan tidak ada informasi yang dikirim ke server eksternal.

5. Keamanan

Karena kami tidak mengumpulkan atau menyimpan data pribadi apa pun, tidak ada risiko informasi Anda terungkap atau dicuri. Game "Game Console City" tidak terhubung ke server atau mengakses jaringan untuk mengirim atau menerima data pengguna.

6. Hak dan Privasi Anda

Anda memiliki privasi penuh saat menggunakan game ini. Tidak ada data pribadi yang dikumpulkan, digunakan, atau disimpan oleh kami, sehingga Anda tidak perlu khawatir tentang pelanggaran privasi atau keamanan informasi.

7. Perubahan Kebijakan Privasi

Kebijakan Privasi ini dapat diperbarui dari waktu ke waktu. Jika ada perubahan, kami akan memberi tahu pengguna melalui pembaruan game. Kami mendorong Anda untuk meninjau Kebijakan Privasi ini secara berkala untuk tetap mendapatkan informasi terbaru.
